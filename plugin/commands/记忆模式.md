---
description: 切换或查看"进度记忆"的记录模式(自动 / 手动)。
argument-hint: 自动 | 手动 | (留空查看)
---

用户输入的参数:`$ARGUMENTS`

请据此操作 progress-memory 的记录模式(默认作用于**当前项目**):
- 参数含 "手动" 或 "manual" → 调用 MCP 工具 `memory_set_mode({mode:"manual"})`
- 参数含 "自动" 或 "auto" → 调用 `memory_set_mode({mode:"auto"})`
- 参数为空 → 调用 `memory_get_mode()` 查看当前模式
- 仅当用户明确说"全局/所有项目"时,调用时显式传 `project:"*"`

完成后用一句话确认:当前项目现在是【自动 / 手动】模式,并简述行为(自动 = 每轮实质改动后我自动记;手动 = 只有 /记进度 时才记,但原始改动始终自动留痕)。
